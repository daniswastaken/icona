# Icona

A collection of icons for Minecraft project descriptions.

## Usage & Attribution

Everyone is free to use these icons in their Minecraft project descriptions. However, as a condition of use, you **must** credit the project by including the following line in your project description:

```text
Credits:
Icona - https://github.com/daniswastaken/icona
```
